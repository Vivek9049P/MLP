# MLP Depth Tutorial

This repository explores how network **depth** affects the performance of **Multilayer Perceptrons (MLPs)** on the **MNIST** dataset.

## Contents
- LICENSE
- MLP.py
- MLP_Depth_MNIST_Report.pdf
- README.md

## Summary
Three MLP models with depths 1, 2, and 4 were trained under identical conditions. Depth-2 gave the best generalisation. Depth-4 showed mild overfitting.

## How to run
```
pip install -r requirements.txt
jupyter notebook notebook/mlp_depth_mnist.ipynb
```
